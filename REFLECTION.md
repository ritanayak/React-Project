Developing the React IP Address Tracker was an enriching experience that strengthened my understanding of both frontend development and API integration. I began by structuring the application into reusable components: SearchForm, InfoPanel, and MapView. This modular approach allowed me to manage state effectively and separate concerns between UI, user input, and data display. Using React hooks, particularly useState and useEffect, I implemented dynamic updates based on user input and window resizing, which was crucial for achieving responsive design.

One of the primary challenges I faced was ensuring the interface worked seamlessly across desktop and mobile devices. Initially, the info section displayed vertically on both views due to CSS grid settings. I solved this by switching the info container to a flex layout for desktop (flex-direction: row) and adjusting it to a column layout for mobile via media queries. Another challenge was integrating LeafletJS for interactive maps while dynamically updating markers and centering based on API data. Handling asynchronous API requests, input validation, and error states required careful attention to avoid runtime issues and improve user experience.

Through this project, I gained hands-on experience in responsive design, asynchronous data fetching, and state management in React. I also improved my skills in debugging layout issues and implementing conditional rendering.

For future improvements, I plan to enhance accessibility, implement additional input validation for domain searches, and add map animations for smoother transitions. I also want to optimize performance by memoizing components and minimizing re-renders when updating state. Overall, this project reinforced my ability to build a fully responsive, interactive, and user-friendly React application.
